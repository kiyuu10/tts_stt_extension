About

Extension information and privacy notes.

TTS & STT Assistant

Version 1.0.0 — Manifest V3

🔒
Privacy

Your API keys are stored locally using chrome.storage.local and are never sent to any server other than Google's APIs. No data is collected by this extension.

🌐
API Calls

All API calls go directly from your browser to Google's endpoints. There is no intermediate backend server.

⚠️
CORS / CSP Notes

The extension uses host_permissions to allow direct calls to Google APIs from the service worker, bypassing typical CORS restrictions. This is safe since only your browser makes these calls.
